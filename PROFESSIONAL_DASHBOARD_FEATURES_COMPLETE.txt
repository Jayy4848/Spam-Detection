CUSTOMIZATION OPTIONS
═══════════════════════════════════════════════════════════════════════════════

1. CHANGE AVATAR LETTER
─────────────────────────────────────────────────────────────────────────────
File: frontend/src/components/Navbar.jsx
Line: <span className="avatar-text">A</span>

Change "A" to any letter or emoji:
<span className="avatar-text">J</span>
<span className="avatar-text">👤</span>

2. CHANGE AVATAR STATUS COLOR
─────────────────────────────────────────────────────────────────────────────
File: frontend/src/App.css
Class: .avatar-status

Change background color:
background: #10b981; /* Green - Online */
background: #f59e0b; /* Orange - Away */
background: #ef4444; /* Red - Busy */
background: #6b7280; /* Gray - Offline */

3. ADJUST GLOW INTENSITY
─────────────────────────────────────────────────────────────────────────────
File: frontend/src/App.css
Animation: @keyframes glow-pulse

Change opacity values:
50% { transform: scale(1.1); opacity: 0.8; } /* More intense */
50% { transform: scale(1.1); opacity: 0.4; } /* Less intense */

4. CHANGE DARK MODE COLORS
─────────────────────────────────────────────────────────────────────────────
File: frontend/src/App.css
Section: body.dark-mode

Modify CSS variables:
--bg: #0f172a; /* Background color */
--surface: #1e293b; /* Card background */
--text-primary: #f1f5f9; /* Main text */

5. DISABLE ANIMATIONS
─────────────────────────────────────────────────────────────────────────────
Add to App.css:
* {
  animation: none !important;
  transition: none !important;
}

Or use prefers-reduced-motion:
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}

PERFORMANCE CONSIDERATIONS
═══════════════════════════════════════════════════════════════════════════════

Optimizations:
• CSS animations use GPU acceleration (transform, opacity)
• Minimal JavaScript (only dark mode toggle)
• localStorage for persistence (no server calls)
• Smooth transitions with cubic-bezier easing
• Efficient selectors and specificity

Performance Impact:
• Dark mode toggle: <1ms
• Theme switch: ~100ms (CSS variable update)
• Animations: 60fps on modern devices
• Glow effects: GPU-accelerated
• No layout thrashing

Best Practices:
• Use will-change for animated properties
• Avoid animating layout properties (width, height)
• Use transform and opacity for animations
• Debounce rapid theme switches if needed
• Lazy load heavy components
