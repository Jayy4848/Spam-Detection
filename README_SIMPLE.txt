        TEXTGUARD SMS SECURITY
     AUTOMATIC SETUP - 3 STEPS ONLY
========================================


STEP 1: RUN THE SCRIPT
========================================

Double-click this file:

    START_HERE.bat

Wait 3-5 minutes. It will:
✓ Find your IP automatically
✓ Configure everything
✓ Build the Android app
✓ Start the server

Keep the window open after it starts!


STEP 2: INSTALL ON PHONE
========================================

The APK file is here:
    android\app\build\outputs\apk\debug\app-debug.apk

Transfer to phone:
- USB cable → Copy to Downloads
- OR Google Drive → Upload & download
- OR Email → Send to yourself

On phone:
1. Enable "Unknown Sources" (Settings → Security)
2. Tap app-debug.apk
3. Tap "Install"


STEP 3: OPEN APP & GRANT PERMISSIONS
========================================

1. Open "TextGuard" app

2. Grant permissions when asked:
   ✓ SMS permissions
   ✓ Notification access

3. Turn ON both toggles:
   ✓ Auto Monitor SMS
   ✓ Notification Analysis

4. Done! Send yourself a test SMS.


THAT'S ALL!
========================================

Every SMS you receive will now be:
✓ Automatically analyzed
✓ Classified (spam/safe/phishing)
✓ Shown in notification
✓ Saved in app feed

No manual work needed!


IMPORTANT:
========================================

✓ Keep the server window open
✓ Both devices on SAME WiFi
✓ Computer must be ON


TROUBLESHOOTING:
========================================

App says "Cannot connect":
→ Check server window is still open
→ Check both on same WiFi
→ Restart server: Close window, run START_HERE.bat again

App not detecting SMS:
→ Grant SMS permissions
→ Turn ON "Auto Monitor SMS" toggle
→ Disable battery optimization for app
