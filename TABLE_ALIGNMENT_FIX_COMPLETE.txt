ISSUE SUMMARY:
--------------
The Dashboard messages table had misaligned columns where data was not appearing 
under the correct headers. Columns like Sentiment, Urgency, Features, Language, 
and Actions were shifting and not properly aligned.

ROOT CAUSE:
-----------
1. Table was using inline styles instead of proper CSS classes
2. No explicit column width definitions using <colgroup>
3. Inconsistent text-align properties across cells
4. Missing cell-specific CSS classes for proper alignment control
