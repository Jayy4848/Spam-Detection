4. NEUMORPHISM EFFECTS
═══════════════════════════════════════════════════════════════════════════════

DESCRIPTION: Soft 3D depth effect (looks like elements are pressed into surface)

VISUAL COMPARISON:

FLAT (Before):
┌──────────────────┐

NEUMORPHISM (After):

  <div className="stat-num">1,234</div>
</div>

VISUAL CHARACTERISTICS:
• Soft shadows (top-left and bottom-right)
• Subtle gradient background
• Pressed-in appearance
• Smooth depth transitions

QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. START THE APP
   cd frontend
   npm start

2. LOOK FOR NEW FEATURES
   ✓ Top-right: User avatar with green status dot
   ✓ Bottom-right: Dark mode toggle button (moon icon)
   ✓ Hover over any icon: See glowing effect
   ✓ Dashboard: Scroll to bottom for performance chart

3. TRY DARK MODE
   • Click moon button (bottom-right)
   • Watch smooth theme transition
   • All colors change to dark theme
   • Click sun button to go back to light mode

4. EXPLORE INTERACTIONS
   • Hover over stat cards: Icons glow and lift
   • Hover over feature tiles: Icons float and glow
   • Hover over buttons: Glow and lift effects
   • Hover over avatar: Scale, rotate, and glow

5. CHECK DASHBOARD
   • Go to Dashboard page
   • Scroll to bottom
   • See "Active ML Model" card
   • Watch animated performance bars

CUSTOMIZATION QUICK TIPS
═══════════════════════════════════════════════════════════════════════════════

Change Avatar Letter:
File: frontend/src/components/Navbar.jsx
Find: <span className="avatar-text">A</span>
Change: A → Your initial

Change Status Color:
File: frontend/src/App.css
Find: .avatar-status { background: #10b981; }
Change: #10b981 → Your color
  Green (#10b981) = Online
  Orange (#f59e0b) = Away
  Red (#ef4444) = Busy
  Gray (#6b7280) = Offline

Adjust Glow Intensity:
File: frontend/src/App.css
Find: opacity: 0.6; (in glow animations)
Change: 0.6 → 0.8 (more intense) or 0.4 (less intense)

Change Dark Mode Colors:
File: frontend/src/App.css
Find: body.dark-mode { --bg: #0f172a; }
Change: #0f172a → Your background color

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Avatar Not Showing:
• Clear browser cache (Ctrl+Shift+R)
• Check console for errors
• Verify Navbar.jsx was updated

Dark Mode Not Working:
• Check if toggle button is visible
• Clear localStorage: localStorage.clear()
• Refresh page
• Check browser console for errors

Glowing Effects Not Visible:
• Hover slowly over icons
• Check if animations are enabled
• Try different browser
• Verify CSS was updated

Performance Chart Not Animating:
• Scroll to bottom of Dashboard
• Wait 1 second for animation
• Check if model data is loaded
• Refresh page

General Issues:
• Clear browser cache
• Restart development server
• Check console for errors
• Verify all files were saved
