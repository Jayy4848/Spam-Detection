========================================
ANDROID APP SETUP (Optional)
========================================

To use automatic SMS monitoring on your phone:

1. Build the Android APK:
   Location: android\app\build\outputs\apk\debug\app-debug.apk
   (If not built yet, run: START_HERE.bat)

2. Transfer APK to phone:
   - USB cable → Copy to Downloads
   - OR Google Drive → Upload & download
   - OR Email → Send to yourself

3. Install on phone:
   - Enable "Unknown Sources" in Settings
   - Tap app-debug.apk
   - Tap "Install"

4. Configure app:
   - Open TextGuard app
   - Tap Settings (⚙️)
   - Set API URL: http://192.168.0.102:8000/api
   - Grant SMS permissions
   - Turn ON "Auto Monitor SMS"

========================================
STOPPING THE SERVERS
========================================

To stop the servers:
1. Press Ctrl+C in each terminal window
2. Or close the terminal windows

To restart:
- Run this command again: "run the project"
- Or manually start:
  Backend: cd backend && python manage.py runserver 0.0.0.0:8000
  Frontend: cd frontend && npm start

========================================
TROUBLESHOOTING
========================================

❌ Can't access from mobile?
   → Check both devices on SAME WiFi
   → Check Windows Firewall (may block port 3000)
   → Run: OPEN_PORT_3000.bat as administrator

❌ Backend not responding?
   → Check Terminal ID: 1 is still running
   → Restart: python manage.py runserver 0.0.0.0:8000

❌ Frontend not loading?
   → Check Terminal ID: 3 is still running
   → Clear browser cache
   → Restart: npm start

❌ API errors?
   → Check backend logs in Terminal ID: 1
   → Verify backend health: http://localhost:8000/api/health/
